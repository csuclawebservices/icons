# Icons
An npm package to use as WordPress Block Editor icon components.

## Install
```bash
npm install @csuclawebservices/icons
```

## Usage
```js
import { Icon } from '@wordpress/icons';
import { layout1 } from '@csuclawebservices/icons';

return (
	<Icon icon={layout1}/>
);
```
